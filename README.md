# React Todo List App

A simple Todo List application built with React.js.

## Features
- Add new tasks
- Delete tasks by clicking on them
- Clean and simple interface

## Technologies Used
- React.js
- Vite
- CSS
- JavaScript (ES6)

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open http://localhost:5173 in your browser

## Project Structure
```
src/
├── App.jsx        # Main app component
├── App.css        # Styles
├── TodoList.jsx   # Todo list component
└── main.jsx       # Entry point
```

## How to Use
- Type a task in the input field
- Click "Add Task" to add it to your list
- Click on any task to delete it
